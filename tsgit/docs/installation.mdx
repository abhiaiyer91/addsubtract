# Installation

## Requirements

- **Node.js**: Version 22.13.0 or higher
- **npm**: Comes with Node.js

<Tabs>
  <Tab title="npm (Recommended)">
    ```bash
    # Clone the repository
    git clone https://github.com/your-username/tsgit.git
    cd tsgit

    # Install dependencies
    npm install

    # Build the project
    npm run build

    # Link globally
    npm link
    ```
  </Tab>
  <Tab title="From Source">
    ```bash
    # Clone the repository
    git clone https://github.com/your-username/tsgit.git
    cd tsgit

    # Install dependencies
    npm install

    # Build
    npm run build

    # Run directly
    node dist/cli.js --help
    ```
  </Tab>
</Tabs>

## Verify Installation

After installation, verify tsgit is working:

```bash
tsgit --help
```

You should see a list of available commands.

## Development Setup

If you want to contribute to tsgit:

```bash
# Clone the repository
git clone https://github.com/your-username/tsgit.git
cd tsgit

# Install dependencies
npm install

# Run in development mode
npm run dev

# Run tests
npm test

# Run tests in watch mode
npm run test:watch
```

## Updating

To update tsgit to the latest version:

```bash
cd tsgit
git pull
npm install
npm run build
```

## Uninstalling

```bash
# Unlink the global command
npm unlink -g tsgit

# Remove the directory
rm -rf /path/to/tsgit
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Command not found after npm link">
    Make sure your npm global bin directory is in your PATH:
    ```bash
    export PATH="$(npm bin -g):$PATH"
    ```
    Add this to your `.bashrc` or `.zshrc` for persistence.
  </Accordion>
  
  <Accordion title="Node.js version too old">
    tsgit requires Node.js 22.13.0 or higher. Update Node.js:
    ```bash
    # Using nvm
    nvm install 22
    nvm use 22
    ```
  </Accordion>
  
  <Accordion title="Permission errors on npm link">
    Try running with sudo or fix npm permissions:
    ```bash
    sudo npm link
    # Or fix permissions
    npm config set prefix '~/.npm-global'
    export PATH=~/.npm-global/bin:$PATH
    ```
  </Accordion>
</AccordionGroup>
